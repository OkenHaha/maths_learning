# Probability and Statistics Syllabus Structure

## Philosophy of the Curriculum

Probability and Statistics should be taught as:

> The mathematics of uncertainty, patterns, information, and decision making

instead of only:
- formulas
- memorization
- calculations

The learner should gradually build intuition about:
- randomness
- uncertainty
- distributions
- patterns in data
- prediction
- inference
- evidence

The curriculum should move from:
- observation
- counting
- experimentation
- uncertainty intuition

towards:
- probabilistic reasoning
- statistical inference
- modeling
- data interpretation

---

# Learning Progression

```text
Observation
→ Uncertainty
→ Counting
→ Probability
→ Randomness
→ Distributions
→ Data interpretation
→ Statistical inference
→ Predictive reasoning
```

# Important Teaching Principle

Teach probability as:

> reasoning under uncertainty

not merely:
- plugging values into formulas

Example progression:
- coin toss intuition
- uncertainty
- outcome spaces
- likelihood
- counting
- probability notation

instead of immediately introducing equations.

---

# Important Design Suggestions

Probability and statistics should heavily use:
- simulations
- visual experiments
- dynamic graphs
- distribution animations
- random event generators
- data visualization tools
- interactive sampling

Visualization and experimentation are essential.

# Most Conceptually Critical Modules

These require especially careful teaching:
- Conditional probability
- Random variables
- Probability distributions
- Inferential statistics
- Hypothesis testing
- Central limit theorem

These determine whether learners:
- truly understand uncertainty and inference
or
- only memorize statistical procedures
