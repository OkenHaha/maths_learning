# Proposed Linear Algebra Syllabus
## Module 0 — Mathematical Foundations

Purpose:
Prepare learners for symbolic reasoning.

Topics:

- Variables and expressions
- Equations and transformations
- Coordinate plane basics
- Functions and graphs
- Arithmetic with negatives and fractions
- Exponents and roots
- Sigma notation basics

Key Goal:
Comfort with symbolic manipulation.

## Module 1 — Introduction to Linear Thinking

Purpose:
Introduce what “linear” actually means visually and intuitively.

Topics:

- What is a linear relationship?
- Straight lines and proportionality
- Slope intuition
- Intercepts
- Linear equations in 2D
- Graphing lines
- Rate of change

Visualization Focus:

movement
direction
scaling
geometric intuition

Core Mental Model:
A linear system transforms space predictably.

## Module 2 — Vectors

This is where real linear algebra begins.

Topics:

- What is a vector?
- Scalars vs vectors
- Vector notation
- Magnitude
- Direction
- Vector addition
- Scalar multiplication
- Unit vectors
- Position vectors

Visualization Focus:
Arrows in space.

Key Intuition:
Vectors represent:

- movement
- direction
- displacement
- change

## Module 3 — Vector Geometry

Topics:

- Distance between vectors
- Midpoints
- Dot product
- Angle between vectors
- Projection
- Orthogonality
- Parallel vectors
- Perpendicular vectors

Visualization Focus:
Geometry of direction.

Key Intuition:
Dot product measures directional similarity.

## Module 4 — Systems of Linear Equations

Topics:

- Solving systems graphically
- Elimination method
- Substitution method
- Matrix interpretation
- Consistent vs inconsistent systems
- Dependent systems

Visualization Focus:
Intersection of lines and planes.

Key Intuition:
Solving equations means finding where constraints meet.

## Module 5 — Matrices

Topics:

- What is a matrix?
- Matrix dimensions
- Matrix notation
- Matrix addition
- Scalar multiplication
- Matrix multiplication
- Identity matrix
- Transpose
- Matrix properties

Visualization Focus:
Matrices as transformations.

Critical Intuition:
A matrix is not just numbers.
It is an operation on space.

## Module 6 — Determinants

Topics:

- Meaning of determinants
- 2×2 determinants
- 3×3 determinants
- Cofactor expansion
- Geometric interpretation
- Area scaling
- Volume scaling
- Singular matrices

Visualization Focus:
Space scaling.

Key Intuition:
Determinants measure transformation scaling.

## Module 7 — Matrix Inverses

Topics:

- What is an inverse matrix?
- Conditions for invertibility
- Finding inverses
- Row reduction
- Solving systems using inverses

Visualization Focus:
Undoing transformations.

Key Intuition:
Inverse matrices reverse operations.

## Module 8 — Row Reduction and Gaussian Elimination

Topics:

- Elementary row operations
- Row echelon form
- Reduced row echelon form
- Gaussian elimination
- Gauss Jordan elimination
- Rank

Visualization Focus:
System simplification.

Key Intuition:
Reduce complexity while preserving meaning.

## Module 9 — Vector Spaces

This is usually where abstraction becomes difficult.

Need very careful teaching.

Topics:

- What is a vector space?
- Subspaces
- Span
- Linear combinations
- Basis
- Dimension

Visualization Focus:
Space generated by directions.

Key Intuition:
Basis vectors build entire spaces.

## Module 10 — Linear Independence

Topics:

- Linear dependence
- Linear independence
- Basis revisited
- Redundancy
- Coordinate systems

Visualization Focus:
Unique directional contribution.

Key Intuition:
Independent vectors add new dimensions.

## Module 11 — Linear Transformations

This is one of the most important conceptual modules.

Topics:

- Transformations
- Rotation
- Reflection
- Scaling
- Shearing
- Transformation matrices
- Composition

Visualization Focus:
Space deformation.

Key Intuition:
Matrices transform geometry.

## Module 12 — Eigenvalues and Eigenvectors

Usually the “mind opening” chapter.

Topics:

- Eigenvectors
- Eigenvalues
- Characteristic equation
- Diagonalization
- Geometric meaning

Visualization Focus:
Invariant directions under transformation.

Key Intuition:
Some directions remain fundamentally unchanged.

## Module 13 — Orthogonality and Projections

Topics:

- Orthogonal bases
- Orthogonal projection
- Gram Schmidt process
- Least squares approximation

Visualization Focus:
Decomposition of space.

Key Intuition:
Projection extracts directional components.

## Module 14 — Advanced Applications
Topics:

- Computer graphics
- Physics
- Machine learning
- Neural networks
- PCA
- Image transformations
- Cryptography
- Recommendation systems

Visualization Focus:
Real world meaning.

Key Intuition:
Linear algebra powers modern computation.

## Module 15 — Computational Linear Algebra

Optional but very useful.

Topics:

- Numerical stability
- Sparse matrices
- Tensor basics
- GPU computation
- Libraries:
	- NumPy
	- PyTorch
	- TensorFlow
